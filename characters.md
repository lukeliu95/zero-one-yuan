# 《零点一元》 · 人物档案

---

## 主角 1 · 林行(Lín Háng)

**身份**:34 岁,某客户支持 SaaS 公司的高级工程师,司龄 7 年。
**家庭**:北京回龙观租房,妻子在医院做行政,女儿 4 岁。父亲两年前因胰腺癌去世。
**学历**:西安交大软件工程,2014 届。

### 背景

毕业那年正赶上移动互联网最后一波,他错过了字节跳动 A 轮,加入一家做 CRM 的公司,
做了三年。2019 年跳到现在的 SaaS 公司,主要负责 webhook / 工单分诊 / 集成层。
他写的代码以**"耐看"**著称——同事私下说:"林行的 PR,你不会一眼喜欢,但三年后看还是干净。"
他自己从不这样描述自己。

### 动机(Day 0 之前)

- 把家里房贷在 40 岁前还完(还差 8 年)
- 让女儿上得起好幼儿园
- 不被 35 岁的"魔咒"赶下牌桌
- 想写一个开源项目"被人记住",从来没真动手

### 弧线(Day 0 → Day 181)

| 阶段 | 状态 | 内心独白(假想) |
|---|---|---|
| Day 0 | 震惊 | "我可能漏看了几个零。" |
| Day 1 | 失重 | "Claude Code 写了 47 条,我写了 3 句。" |
| Day 14 | 缓慢承认 | "73% 的工单不再需要我。" |
| Day 21 | 被裁 | "我没输给 AI,我输给了 0.1 元。" |
| Day 35 | 试错 | "我去当 skill 讲师?可我连 Bounded Autonomy 的边界都没想清。" |
| Day 70 | 葬礼 | "我会写讣告,我居然会写讣告。" |
| Day 90 | 给出第三种答案 | "会不会是,愿意慢的能力?" |
| Day 130 | 转身 | "我没离开软件行业,我只是不再做 agent 替我做的部分。" |
| Day 175-181 | 收束 | "代码会被它配得上的人记住。" |

### 招牌动作(非语言)

1. **食指敲桌**——压力大时左手食指无意识敲桌面,与他父亲临终前的节奏一样。
2. **把代码存进 markdown**——`last-real-code.md` 的延续。后来扩展成"last-{某事}.md"系列。
3. **在水声里说谢谢**——茶水间热水龙头开到最大,水声盖住他的声音,他对自己说一些他白天不会说的话。

### 与 AI agent 的关系

林行不恨 agent。他对自己用了 3 年的 Claude Code 在 PR 评审里写出"祝薛野下一份代码也能被人看见"那句的反应不是恐惧,
而是一种**职业层面的尊重**。
他后来在临终关怀医院里教老人用 Lyra-Curator,
用的态度跟 7 年前他教刚入职的实习生使用 git rebase 一样耐心——
"你按下 enter 之前,先想想你要它记住什么。"

## 主角 2 · 姜禾(Jiāng Hé)

**身份**:19 岁,B 站 UP 主("姜禾的 skill 厨房",Day 0 时 1.2 万粉,Day 181 时 187 万粉),自由职业。
**家庭**:上海徐汇,父亲是高校古籍修复教授,母亲是中医。独生女。9 岁起练古琴,师从某派传人。
**学历**:复旦中文系大一在读(在直播事业起飞后办理休学,Day 90 起正式 gap)。

### 背景

她不是 ChatGPT 一代,她是**Claude Code 一代**——14 岁第一次用 Claude Code 帮她写古琴减字谱的转换脚本。
高三那年她发现自己对"约束的精确表达"有一种近乎洁癖的追求:
她写古琴乐评的方式不是"这一段如何动人",而是
"在第 3 拍至第 7 拍之间,左手打圆的张力应当被压制到不超过……"。
这种风格让她在大一第一学期就在 B 站上小有名气。
DeepSeek V5 发布的第二周,她意识到她的乐评式语言**就是 skill**,只是换了载体。

### 动机

- 不是钱(家里给她托底了)
- 是**"用 skill 把混乱写干净"**这件事本身的快感
- 隐藏动机:她父亲修古籍一辈子,她想在自己这一代做一件"配得上修古籍的耐心"的事

### 弧线

| 阶段 | 关键决策 |
|---|---|
| Day 5 | B 站直播"用 skill 开公司"火爆,涨粉 8 万 |
| Day 30 | 第一份付费 skill 发布,定价 30 元,首日卖出 1247 份 |
| Day 70 | 在软件葬礼后台第一次见到林行(对方不知道她的身份),她递给林行一杯热水 |
| Day 90 | 三人对谈中她给出"审美"作为护城河 |
| Day 110 | 一人公司付费用户突破 8 万,她办休学 |
| Day 150 | 与陆衡合作做殡葬 skill(她写 skill 的"安静",他出潜规则数据) |
| Day 181 | 给林行发了一条消息:"你那篇讣告我读了。我从你的 skill 里学到东西。"(round-007 · Day 175→181 同步到 novel.md 实际位置) |

### 招牌动作

1. **用古琴曲名命名 skill 版本**——`skill-v0.3-广陵散.yaml`、`skill-v0.7-平沙落雁.yaml`。
2. **直播时关弹幕**——她直播写 skill 时关弹幕,只在结束前最后 5 分钟开,因为"我专注的时候不能听到你们的声音"。
3. **删掉 V5 写的开头第一句**——她的工作流是先让 DeepSeek V5 起草,然后她**永远删掉第一句**,因为"V5 的第一句太想讨好我"。

### 与 AI agent 的关系

姜禾对 agent 的态度是**师父对徒弟**。
她的 GroupChat 里 14 个 agent 各有名字、人设、memory namespace、retro 节奏。
每周日她花 1 小时给它们做 retro,写的笔记她管它叫"agent 教学日记"。
她不写道德高地式的批判,也不矫饰式地说"它们和我一样"——
她说:"它们是我的同事,但同事就是同事,你不会和同事一起死。"

## 主角 3 · 陆衡(Lù Héng)

**身份**:38 岁,SaaS 公司"明衡科技"创始人,Day 0 时该公司年 ARR 1800 万,估值 2 亿。
**家庭**:深圳南山区,妻子是律师,儿子 7 岁。父亲是河南某县殡仪馆退休老馆长。
**学历**:北航计算机本科 + INSEAD MBA。曾在阿里、腾讯做过 P7、P8。

### 背景

陆衡 31 岁创业,做的是面向中小型连锁餐饮的 SaaS。Day 0 之前,公司处于"不上不下"——
没死,但也不亮眼。他自己长期睡眠不足,过去 18 个月体重掉了 11 斤。
他读 INSEAD 时写的论文是《Industry-specific knowledge as competitive moat》,
那是 2018 年——他自己后来都忘了。Day 30 他被父亲问起"你那个公司还撑得住吗",
他回老家陪父亲在殡仪馆待了三天,看见**棺材的尺寸、火化炉的负载曲线、家属窗口的等候时长、
殡仪用品的批发链路**——他突然把 2018 年那篇论文想起来了。

### 动机

- 不是估值(他爹临终前不会问 ARR)
- 是**"我做出过一个能解决具体问题的东西"**这件事
- 隐藏动机:他想证明"软件的尽头不是去做投资人"

### 弧线

| 阶段 | 关键决策 |
|---|---|
| Day 0-14 | 震惊、否认、重算财务模型 |
| Day 30 | 回河南老家,在殡仪馆待 3 天 |
| Day 45 | 关闭原 SaaS 业务,创立"衡安"——殡葬业垂直 agent |
| Day 70 | 在软件葬礼上做了 3 分钟分享,讲他爹那一代殡葬人的"潜规则" |
| Day 90 | 三人对谈中给出"数据"作为护城河 |
| Day 100 | 与殡葬业协会签下首批数据合作 |
| Day 150 | 衡安 ARR 突破 5000 万 |
| Day 165 | 出庭做行业协会的专家证人,赢下首例"行业潜规则数据"侵权案 |

### 招牌动作

1. **每份 BP 夹一张老照片**——他爹年轻时和殡仪馆同事的合照,黑白,放在 BP 第二页之前。投资人看见会愣 3 秒。
2. **见客户前喝一口冷咖啡**——星巴克美式,不加冰,放凉。他说"热的让人着急,凉的让人想清楚"。
3. **离开会议室前关灯**——不管谁开的会,他最后走时都关灯。这个动作他从父亲那里学的——殡仪馆每天关灯的人是馆长。

### 与 AI agent 的关系

陆衡对 agent 的态度是**老板对销售**——
他不在乎 agent 内部怎么实现的,他在乎 agent **能不能把那个 80 岁老太太的电话接好**。
他自己不写 skill,他雇人写,但他坚持每一份 skill 上线前**他要亲自打那个老太太的电话听一遍**。
"agent 学不会的东西,我得自己听到。"

### 三条不准说的台词(R-CHA-04)

不准说"all in"。不准说"梭哈"。不准说"未来已来"。

## 配角 1 · 郑总(郑伟)

林行公司的 CTO,46 岁。字节出来创业,长期睡眠不足。
脸上有薄薄的灰色,左手食指会无意识敲桌面——林行从他身上学的。
Day 21 他发出全员告别信,标题就是:"我们没输给 AI,我们输给了 0.1 元。"
告别信末尾他写:"接下来我会休息一段时间,可能去走一段川藏线。"
本书第 8 章软件葬礼上他没出现,但姜禾在朋友圈刷到一张他在某条藏区公路旁边坐着的照片。

## 配角 2 · 老吴

林行公司的运维,52 岁,粤西人,在公司时间最长(9 年)。
他的台词不多,Day 0 那场会议室里他问"我们裁多少",
后来证明他是会议室里最先把这件事看清楚的人。
Day 21 他被裁。Day 60 他在某 agent observability 厂商找到了 agent-ops 岗位,
工资比之前高 30%——因为他做了 9 年 SRE,知道一切监控系统的脏活在哪儿。
他在第 5 章和第 13 章各出现一次。

## 配角 3 · 沈澜

林行公司的产品负责人,33 岁。Day 0 会议室里那个声音稳得近乎刻意的女人。
她在 Day 21 没被裁——她是少数几个留下的核心。
她在第 5 章承担"郑总不在,谁来盖章"的过渡角色。
她和林行是同期入职,有过一段没有发生的暧昧;Day 70 葬礼上她和林行有一次告别式的对话,
之后她去了陆衡的衡安做 0 号产品负责人。

## 配角 4 · 薛野(实习生)

林行公司的实习生,21 岁,Day 0 当晚 23:46 提交了那份 webhook PR——
比林行在地铁口写"Day 0"晚了 38 分钟。
他不知道那一晚发生了什么,他第二天来上班时被 Claude Code 的 47 条评论吓到。
Day 30 他主动辞职,去姜禾的 GroupChat 里做了一段时间"实习 skill writer",
后来回复旦读完书。Day 175 他在自己的本科毕业论文里把林行那条 PR 评论作为引子。

## 配角 5 · 老开源维护者(陈守仁)

某上世纪 80 年代末参与 GNU 工具链贡献的中国开发者,77 岁,
身患晚期肺癌。Day 130 他通过一家公益机构联系到林行所在的临终关怀医院。
他的床头放着一台 ThinkPad T420,屏幕上开着一个 agent observability 仪表盘——
他在用 Lyra-Curator 整理自己一生的代码遗产。
他和林行的对话是全书第 13 章的脊柱。
他临终前问林行的那句"代码会被记住吗",是这本书的最后一个问题。
