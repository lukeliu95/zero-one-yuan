# Round 004 · 零点一元 · 7 章密度扩写(prose polish)

## 1. 本轮目标

用户原话(摘要):把 round-003 已写但字数偏短的 7 章——第 5、6、7、8、10、11、12 章——从当前 ~1500-1800 中文字扩到 **2500-3000 中文字**,通过新增**场景细节、配角侧写、内心活动留白、感官描写**,而**不**通过填充对话或概念科普。

硬约束:
- 不重写第 1、2、3、4、4.5、9、13 章
- 不新增主线情节,只在每章已有骨架内深化
- 不追加新概念锚点(R-CON-02 守住)
- 不破坏冷克制风格
- 不动章末收尾(每章 "(第 N 章 · 完)" 之前的最后一段保留 round-003 原文)

## 2. 决策

### 2.1 扩写策略

每章选 2-4 个"原有段落之间的缝"插入新段落。**不在章末追加**,**不替换原段落**(只 add,不 replace),保证 round-003 已通过的所有元素 100% 保留。

每章新增内容由四类构成,按比例混合:

1. **感官描写**(气味 / 声音 / 触觉)— 每章 1-2 段
2. **配角侧写**(沈知微的火候 / 驼色西装学员 / 陆衡他爹的姿势 / 陈守仁讲妻子的六秒 / 白护士长试温 / 沈澜笔记本)— 每章 1-2 处
3. **招牌动作复用一次**(林行食指敲桌 / 姜禾摸琴缝 / 陆衡关灯)— 每章 ≥ 1
4. **代际 / 时间感锚定**(1998 下岗 / 1989 GCC / 1979 当兵 / 80 年代灯管)— 每章 1 处

### 2.2 守住的边界

- 风格:长短句肌肉记忆延续前 4 章
- 概念:本轮 0 新增概念,只让已有概念被更密的场景使用
- 角色:三主角招牌动作零漂移,陆衡台词三禁(all in / 梭哈 / 未来已来)继续 0 出现
- 字数策略:目标 2500-3000,允许下沿 -50 字弹性
